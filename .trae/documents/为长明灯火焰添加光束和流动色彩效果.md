# 实现计划

## 1. 增强火焰的光束效果
- 为 `.flame`、`.flame::before` 和 `.flame::after` 元素添加线性渐变背景，模拟从中心向外扩散的光束
- 增强 `filter` 属性，添加更多的 `drop-shadow` 效果，创造更明显的光束扩散
- 调整 `box-shadow` 属性，添加内部发光效果，增强光束的层次感

## 2. 创建流动色彩动画
- 创建 `cyber-flame-beam-flow` 关键帧动画，实现光束的流动效果
  - 通过调整 `background-position` 实现渐变的流动
  - 添加 `transform` 变换，实现光束的摇摆效果
- 创建 `cyber-flame-color-shift` 关键帧动画，实现色彩的动态变化
  - 调整 `filter` 中的 `hue-rotate` 属性，实现色相的循环变化
  - 调整 `box-shadow` 的颜色值，实现色彩的流动变化

## 3. 集成新动画到现有动画系统
- 将新的动画添加到 `.lightLampWrapper.lit .flame` 的动画列表中
- 为点燃状态添加相应的光束和色彩变化动画
- 确保新的动画与现有的动画协调工作，不产生冲突

## 4. 优化动画性能
- 确保动画使用硬件加速属性（`transform` 和 `opacity`）
- 合理设置动画的持续时间和缓动函数，确保流畅的视觉效果
- 避免过多的动画同时运行，确保页面性能不受影响

## 5. 保持其他元素不变
- 只修改火焰相关的CSS代码
- 不修改HTML结构或其他组件
- 确保页面的整体布局和样式保持不变

## 关键技术点
- 使用 CSS 线性渐变和径向渐变创建光束效果
- 使用 CSS `filter` 属性实现发光和色彩变化
- 使用 CSS 动画关键帧实现动态效果
- 使用 CSS `transform` 属性实现硬件加速动画

这个计划将确保长明灯的火焰具有明显的光束和流动色彩的动态变化，同时保持页面的其他元素不变。
