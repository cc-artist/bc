## 问题分析
后端管理界面 `admin.html` 无法预览，返回"接口不存在"错误。通过检查代码发现，`backend/src/index.js` 文件中缺少静态文件服务配置，导致 Express 服务器无法处理 HTML 文件请求。

## 解决方案
在 `backend/src/index.js` 中添加静态文件服务中间件，具体修改如下：

1. 引入 `path` 模块，用于处理文件路径
2. 添加 `express.static` 中间件，将 `backend` 目录设为静态文件根目录
3. 确保静态文件中间件在路由配置之前添加，优先处理静态文件请求

## 代码修改
```javascript
// 在文件顶部添加 path 模块引入
const path = require('path');

// 在中间件配置部分添加静态文件服务
app.use(express.static(path.join(__dirname, '../')));
```

## 预期效果
- 访问 `http://localhost:3001/admin.html` 可以正常预览后端管理界面
- 静态文件服务会优先处理 HTML、CSS、JS 等文件请求
- API 路由不受影响，仍正常工作

## 测试方法
1. 重启后端服务器
2. 在浏览器中访问 `http://localhost:3001/admin.html`
3. 验证管理界面是否正常加载并可以使用所有功能
