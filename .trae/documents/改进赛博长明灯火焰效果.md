## 问题分析
当前赛博长明灯的火焰效果存在以下问题：
1. **形状不自然**：使用旋转45度的扇形设计，缺乏火焰的自然形态
2. **色彩过渡生硬**：从白色到红色的渐变不够流畅
3. **动画效果简单**：只有基本的闪烁效果，缺乏自然摇曳感
4. **赛博风格融合不足**：传统火焰效果与长明灯的霓虹边框、科技纹理不协调

## 改进方案
### 1. 赛博朋克火焰形状设计
- **核心形状**：使用多层叠加的不规则多边形，模拟火焰的自然形态
- **赛博元素**：添加霓虹边框和发光效果
- **实现方式**：使用CSS `clip-path` 或 `border-radius` 组合创建复杂形状

### 2. 霓虹色彩系统调整
- **主色调**：融合青色（#00ffff）、紫色（#ff00ff）与传统火焰色
- **渐变层次**：
  - 中心：白色高光
  - 中层：金色+青色渐变
  - 外层：紫色+红色渐变
- **发光效果**：添加多层 `box-shadow`，增强赛博发光感

### 3. 高级动画系统
- **火焰摇曳**：使用 `transform: rotate()` 和 `translate()` 实现自然摆动
- **呼吸效果**：调整透明度和大小，模拟火焰的呼吸感
- **脉冲效果**：添加快速的亮度变化，增强赛博科技感
- **图层动画**：不同火焰图层使用不同频率的动画，创造层次感

### 4. 状态管理优化
- **未点燃状态**：几乎透明，只有微弱的轮廓
- **点燃中状态**：火焰逐渐增大，亮度逐渐增强
- **已点燃状态**：完整的赛博火焰效果，所有动画激活

## 实现步骤
1. 修改 `.flame` 类的基本形状和色彩
2. 重写火焰动画，添加摇曳和呼吸效果
3. 调整火焰的状态变化逻辑
4. 添加赛博朋克元素，如霓虹边框和发光效果
5. 测试不同状态下的火焰表现

## 预期效果
- 火焰形态更自然，具有赛博朋克风格
- 动画效果丰富，具有自然摇曳感和科技脉冲感
- 与长明灯的其他赛博元素协调统一
- 状态变化流畅，增强用户体验

## 技术要点
- 使用 CSS `clip-path` 创建复杂火焰形状
- 实现多图层动画效果
- 融合赛博朋克霓虹色彩系统
- 优化性能，确保动画流畅运行
