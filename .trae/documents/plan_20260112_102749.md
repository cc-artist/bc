# 实现方案

## 核心需求
1. "南无阿弥陀佛"窗口所有显示的文字均以中、英文显示
2. 独立一行英文在上，独立一行中文在下
3. 包括标题、祝福语和日期
4. 保持原有的神秘风格和布局

## 实现步骤

### 1. 创建英文祝福语库
- **文件**: `src/app/page.tsx`
- **修改内容**: 
  - 创建一个与中文古文字祝福语对应的英文祝福语数组
  - 确保每个中文祝福语都有对应的英文翻译
  - 保持祝福语的准确性和神圣感

### 2. 修改祝福语显示逻辑
- **修改内容**: 
  - 修改`getRandomQuote`函数，同时返回英文和中文祝福语
  - 调整状态管理，存储当前的英文和中文祝福语
  - 修改JSX结构，实现中英文分行显示，英文在上，中文在下

### 3. 修改日期显示逻辑
- **修改内容**: 
  - 添加英文日期显示
  - 实现日期的中英文分行显示
  - 确保日期格式正确，英文在上，中文在下

### 4. 调整CSS样式
- **文件**: `src/app/page.module.css`
- **修改内容**: 
  - 确保中英文文字都使用合适的字体
  - 调整行高和间距，确保文字美观易读
  - 保持文字阴影效果，增强神秘感

## 代码修改要点

1. **英文祝福语库**: 创建与中文古文字对应的英文祝福语数组
2. **状态管理**: 调整现有状态，存储中英文祝福语
3. **JSX结构**: 修改文字显示结构，实现中英文分行显示
4. **CSS样式**: 调整样式，确保中英文文字美观协调

## 预期效果
当用户打开"南无阿弥陀佛"窗口时，会看到：

- 日期：英文在上，中文古文字在下
