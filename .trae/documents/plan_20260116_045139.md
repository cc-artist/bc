## 修复长明灯火焰点燃过程不展示问题

### 问题分析

长明灯火焰从开始点燃到燃烧的变化过程没有展示出来，经过分析发现问题出在 CSS 样式定义上：

1. 在 `.lightLampWrapper.lighting .flame` 样式中，直接设置了 `transform` 属性，覆盖了 `cyber-flame-ignite` 动画中的初始 transform 值
2. 同时设置了 `transition` 属性，与动画产生冲突，导致动画无法正常播放
3. 火焰点燃过程的动画被覆盖，所以用户看不到从开始点燃到燃烧的变化过程
4. 动画优先级设置不正确，导致多个动画之间产生冲突

### 解决方案

1. **修复 CSS 样式冲突**
   - 移除 `.lightLampWrapper.lighting .flame` 中的 `transform` 属性，让 `cyber-flame-ignite` 动画完全控制火焰的变换
   - 移除 `.lightLampWrapper.lighting .flame` 中的 `transition` 属性，避免与动画产生冲突
   - 确保动画能够不受干扰地执行，展示完整的点燃过程

2. **优化动画优先级**
   - 调整动画的执行顺序，确保 `cyber-flame-ignite` 动画能够正确执行
   - 确保 `forwards` 属性能够正确生效，保持动画结束状态

3. **增强视觉反馈**
   - 确保点燃过程的每个阶段都有明显的视觉变化
   - 增强火焰动画的可见性，确保用户能够清晰看到点燃过程

### 修改文件
