## 启动前后端服务计划

### 1. 启动后端服务
- **目录**: `cyber-buddha-blessing/backend`
- **命令**: `npm run start`
- **端口**: 5000
- **目的**: 提供图片合成服务，处理用户上传并添加佛光效果

### 2. 启动前端服务
- **目录**: `cyber-buddha-blessing`
- **命令**: `npm run dev`
- **端口**: 3000
- **目的**: 提供用户界面，允许上传图片并查看开光结果

### 3. 验证服务状态
- 检查后端服务是否在端口5000正常运行
- 检查前端服务是否在端口3000正常运行
- 确认服务间通信正常

### 4. 确认功能完整性
- 验证图片合成功能是否正常工作
- 确认物品图片边缘有佛光效果
- 确认物品图片上下浮动，背景图静止
