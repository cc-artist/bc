## 问题分析

目前，"点亮长明灯祈福"功能已经实现了基本的模态框显示和音频播放，但缺少用户交互体验，特别是用户点击长明灯点燃火焰的过程和伴随的音乐效果。

## 解决方案

### 1. 添加状态管理

**修改 `src/app/page.tsx` 文件：**

- 添加 `lightState` 状态，用于控制长明灯的点燃状态：
  - `'unlit'` - 未点燃
  - `'lighting'` - 点燃中
  - `'lit'` - 已点燃

### 2. 修改长明灯实现

**修改 `src/app/page.tsx` 文件：**

- 给长明灯图片添加点击事件，触发点燃过程
- 修改音频播放逻辑，只播放约5秒钟
- 添加火焰点燃的动画效果
- 控制祝福语句的显示时机，在火焰点燃后显示

### 3. 添加CSS动画

**修改 `src/app/page.module.css` 文件：**

- 添加火焰点燃的动画效果
- 添加长明灯点燃状态的样式
- 添加祝福语句的显示动画

## 预期效果

修改后，"点亮长明灯祈福"功能将实现以下效果：

1. 用户点击长明灯图片
2. 长明灯开始点燃过程
3. 播放约5秒钟的菩萨念心经音乐
4. 音乐停止后，长明灯火焰被点燃烧旺
5. 显示祝福语句

## 实现步骤

1. 修改 `src/app/page.tsx` 文件，添加 `lightState` 状态
2. 给长明灯图片添加点击事件，触发点燃过程
3. 修改音频播放逻辑，只播放约5秒钟
4. 添加火焰点燃的动画效果
5. 控制祝福语句的显示时机
6. 在 `src/app/page.module.css` 文件中添加火焰点燃的动画效果
7. 测试功能是否正常工作

## 技术细节

- 使用 React 的 `useState` 管理长明灯的点燃状态
- 使用 React 的 `useEffect` 控制音频播放时间
- 使用 CSS 动画实现火焰点燃效果
- 使用 `setTimeout` 控制音乐播放时长和祝福语句的显示时机

这个实现计划将为用户提供一个交互式的长明灯点燃体验，符合用户的需求。
